Цели и задачи проекта: 

Цель проекта - провести разведывательный анализ данных об условиях жизни учащихся в возрасте от 15 до 22 лет и их успеваемости  по математике для последующего построения модели, которая предсказывала бы результаты госэкзамена по математике для каждого ученика школы.

Задачи проекта: 

1. Провести первичную обработку данных.
2. Посмотеть на распределение признака для числовых переменных, устранить выбросы.
3. Оценить количество уникальных значений для номинативных переменных.
4. По необходимости преобраховать данные
5. Провести корреляционный анализ количественных переменных
6. Отобрать не коррелирующие переменные.
7. Проанализировать номинативные переменные и устранить те, которые не влияют на предсказываемую величину (в нашем случае — на переменную score).
8. Сформулировать выводы относительно качества данных и тех переменных, которые будем использовать в дальнейшем построении модели.

Отчет содержит 4 основных раздела: 
1. Первичный анализ данных. Числовые переменные
2. Первичный анализ данных. Номинативные переменные
3. Корреляционный анализ
4. Анализ значимости по тесту Стъюдента
Каждый раздел сопровождается промежуточными комментариями и выводами. 

В конце отчета также имеются итоговые выводы и дополнительные замечания.
