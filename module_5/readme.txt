Описание проекта: 

Цели и задачи проекта: 

Цель проекта - провести сбор и анализ данных об автомобилях и на их основе создать модель, предсказывающую стоимость автомобиля по его характеристикам.


Задачи проекта: 

0. Провести сбор данных с сайта auto.ru
1. Провести первичную обработку данных.
2. Посмотреть на распределение признака для числовых переменных, устранить выбросы.
3. Оценить количество уникальных значений, заполнить пропуски
4. По необходимости преобразовать данные
5. Сгенерировать новые признаки
6. Создать модели, попробовать подобрать их параметры, оценить их качество
7. Сделать Ансамбль моделей, Blending, Stacking


Модель необходима для быстрого выявления выгодного предложения (когда желаемая цена продавца ниже предсказанной рыночной цены).  
Сбор данных был произведен с сайта auto.ru. Для оценки качества решения использовалась метрика MAPE (Mean Percentage Absolute Error).
